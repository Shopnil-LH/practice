## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Custom Path Shortcuts

- **@src/\*** : ./src folder
- **@pages/\***: ./pages folder
- **@styles/\*** : ./styles folder
- **@root/\*** : root directory
